Questions

  1. Answer the following questions about the tree shown below: 

            A
       / \
      B   C
     /   /|\
    D   E F G
   /|    / \
  H I   J   K
    1. How many nodes does this tree have? how many edges?
    2. Which node is the root of this tree?
    3. Which node is the parent of node B?
    4. Which nodes are children of node C?
    5. Which nodes are leaves of the tree?
    6. What is the depth of node D?
    7. What is the height of the tree?
    8. Give the nodes along a path from node F to node B. How long is this path?
    9. Which nodes are the descendants of node C?

  2. Consider the following incomplete code fragment: 

    
int[] arr = {6, 7, 1, 3, 5, 8, 2, 4};
STATEMENT1

for (int i = 0; i < arr.length; i++) {
    STATEMENT2
}

while (COND) {
    System.out.print(EXPR + ", ");
}

    For parts (a) through (c), give the output of the above code if the underlined expressions are completed as given. Assume that Stack is implemented using a linked list, Queue is implemented using an array, and PriorityQueue is implemented using a heap.

    1. STATEMENT1:Stack<Integer> s = new Stack<Integer>();
      STATEMENT2:s.push(arr[i]);
      COND:!s.isEmpty()
      EXPR:s.pop()
    2. STATEMENT1:Queue<Integer> q = new Queue<Integer>();
      STATEMENT2:q.enqueue(arr[i]);
      COND:!q.isEmpty()
      EXPR:q.dequeue()
    3. STATEMENT1:PriorityQueue<Integer> pq = new PriorityQueue<Integer>();
      STATEMENT2:pq.enqueue(arr[i]);
      COND:!pq.isEmpty()
      EXPR:pq.dequeue()
    4. Suppose we replace arr with an array of N integers. For each of the above three cases, give the worst-case time complexity (Big-O) of the code fragment. (Neglect the time required to create arr, and assume System.out.print takes constant time.)

    1. Using the same format as in part (b), show the array corresponding to the following heap represented as a tree: 
              9
      /   \
    4       8
   / \     / \
  3   2   5   7
 / \
1   3
    2. Using the same format as in part (a), show the tree corresponding to the following heap represented as an array (the array is 0 indexed): 
      0, 9, 7, 8, 4, 3, 7, 5, 2
       Only use space characters in your answer. Do not use tab characters since they may result in incorrect spacing.
    3. Show the result (as an array) of running enqueue(8) on the heap in part (a).
    4. Show the result (as an array) of running enqueue(10) on the heap in part (b).
    5. Show the result (as an array) of running dequeue() on the heap in part (a).
    6. Show the result (as an array) of running dequeue() on the heap in part (b).
